# Vue-Admin-Demo

> 这是一个基于vue2.0 + elementUI 后台管理平台

### 技术栈
- 传说中的VUE全家桶(vue vue-router vuex)
- axios
- [Element UI](http://element-cn.eleme.io/#/zh-CN)
- [Mock](http://mockjs.com)（生成随机数据，屌屌的）
